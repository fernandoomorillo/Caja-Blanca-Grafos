# PruebasCajaBlanca
El proyecto es un juego de tablero al azar, en el que tiras un dado y vas avanzando casillas hasta llegar al final o también puedes caer en casillas que te pueden perjudicar

## Tecnologías utilizadas
- Draw.io
- Eclipse
- JAVA

## Deploy
Abrimos el proyecto de guiaviajero en eclipse y ejecutamos el código
